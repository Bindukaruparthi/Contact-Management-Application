Task 1 – Contact Management Application

Candidate Name: Karuparthi Bindu Padmavathi

Technology Stack Used:

* Python
* Streamlit (Frontend + Backend)
* SQLite (Database)

Project Description:
This application is a Contact Management System that allows users to perform CRUD operations:

* Create new contacts
* View all contacts
* Update existing contacts
* Delete contacts

Features:

* User-friendly interface
* Data validation (email format, numeric phone)
* Prevention of duplicate email entries
* Persistent data storage using SQLite database

How to Run the Application:

1. Install dependencies:
   pip install streamlit

2. Run the application:
   streamlit run app.py

3. The application will open in the browser automatically.

Challenges Faced:

* Handling duplicate email entries, resolved using unique constraints in database
* Implementing structured UI for smooth CRUD operations
